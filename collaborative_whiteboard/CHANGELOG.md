# Changelog

All notable changes to this project will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.0.0/),
and this project adheres to [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

## [1.0.0] - 2023-10-30

### Added
- Initial release of the Collaborative Whiteboard application
- Real-time multi-user whiteboard collaboration
- Authentication with email/password and Google sign-in
- Drawing tools: freehand, line, rectangle, circle, text, and eraser
- Session management (create and join sessions)
- Color picker with customizable colors
- Zoom in/out and pan functionality
- Undo/redo capability
- Clear board option
- Participant list in active sessions
- Dark/light mode support
- Firebase Realtime Database integration for live syncing
- Splash screen and app icon

### Changed
- None (initial release)

### Fixed
- None (initial release)

## [0.1.0] - 2023-09-15

### Added

- Basic drawing functionality
